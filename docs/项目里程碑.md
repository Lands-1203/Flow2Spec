## M1 · CLI 初始化与 OpenSpec 协作流

- **交付**：CLI 具备初始化与可安装的产品雏形；早期围绕 OpenSpec / opsx 变更流组织需求、变更和执行。
- **依据**：commit `009a9d6`（初始化）、`abd0c37`（版本更新）；`.Knowledge/stock-docs/Flow2Spec-项目里程碑.md` 的 M0 / M1；早期 tag 序列至 `V2.2.3`。
